# 🛡️ Local AI Code Auditor

Hackathon project done in 8 hours. The theme was an app that could provide fixes, security analysis, performance analysis, summary etc. for code through a locally hosted LLM.
The project was developed, as suggested by the organizers, by using AI tools and assistance, to achieve high-velocity delivery.

## 🌟 Key Features

- **Local Inference:** Connects to local models via Ollama.
- **In-depth Analysis:** Scans for vulnerabilities, performance issues, code errors etc.
- **Auto-Fix Engine:** Automatically generates and applies patches for identified errors and performance bottlenecks.

## 🏗️ Technical Stack

- **Frontend:** Avalonia UI
- **Backend:** C#
- **LLM Orchestration:** Ollama
